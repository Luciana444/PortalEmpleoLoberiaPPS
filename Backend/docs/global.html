(constant) transporter :Object

Configuración del transporte de correo electrónico mediante Nodemailer. Este objeto define cómo se enviarán los correos electrónicos salientes del sistema, utilizando un servidor SMTP autenticado. - Por defecto se utiliza Gmail (`smtp.gmail.com`), pero puede adaptarse a otro servidor. - Usa TLS (STARTTLS) en el puerto 587. - Las credenciales y parámetros se leen desde las variables de entorno: - `SMTP_HOST` - `SMTP_PORT` - `SMTP_USER` - `SMTP_PASS`
